### new_int_array
- 入参：`size`：`int` 数组长度
- 函数说明，创建`int`类型的数组
```js

```

### new_array
- 入参：`Class`：类型
- 入参：`size`：`int` 数组长度
- 函数说明，创建`Object`类型的数组
```js
return new_array(1); // [null]  // Object 类型的数组
return new_array(String.class, 1); // [null] String类型的数组
```
