启动项目前操作：

后端：
* 创建数据库，字符集：`utf8mb4`，排序规则：`utf8mb4_unicode_ci`
* 导入`db`文件夹`magic-boot.sql`文件
* `MagicBootApplication`运行即可

前端（`magic-boot-ui`）：  
```
nodejs要求版本 >= 14
```
执行命令
```
npm install
npm run dev
```
浏览器输入：`http://localhost:3000`  
用户名密码：`admin/123456`
